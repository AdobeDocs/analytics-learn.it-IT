# Rieseguire un processo di feed dati

L’interfaccia utente del feed dati in Adobe Analytics semplifica notevolmente la gestione del feed dati. Se hai accesso alla gestione dei feed dati, puoi eseguire nuovamente un processo di feed dati autonomamente nell’interfaccia utente dei feed dati, senza che il team di assistenza clienti invii nuovamente i file di feed dati.
