# Scaricare la chiave di autenticazione del feed dati

La creazione di un feed di dati consente ad Adobe di sapere dove inviare i file di dati non elaborati e cosa si desidera includere in ciascun file. Il supporto SFTP per i feed di dati è disponibile in Analytics. Richiede un host SFTP, un nome utente e il sito di destinazione per contenere una chiave pubblica RSA o DSA valida. Puoi scaricare la chiave pubblica appropriata al momento della creazione del feed.
