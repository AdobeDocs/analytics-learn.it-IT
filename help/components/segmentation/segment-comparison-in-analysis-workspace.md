# [!UICONTROL Segment] Confronto in Analysis Workspace {#segment-comparison-in-analysis-workspace}

Comprendere le vere differenze tra i segmenti di marketing è sempre stato un processo difficile e lungo. [!DNL Segment IQ], parte di Analysis Workspace in Adobe Analytics, semplifica questa analisi esaminando due  [!DNL Analytics] [!UICONTROL segments] tra tutte le tue risorse  [!UICONTROL dimensions] e  [!UICONTROL metrics] per scoprire automaticamente le loro differenze più importanti dal punto di vista statistico.

>[!VIDEO](https://video.tv.adobe.com/v/23976/?quality=12)

Per ulteriori informazioni su questa funzione, consulta la [documentazione](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/panels/segment-comparison/segment-comparison.html?lang=en).
