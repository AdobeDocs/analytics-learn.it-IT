# [!UICONTROL Activity Map] in Adobe Analytics {#activity-map-in-adobe-analytics}

[!UICONTROL Activity Map] è uno strumento disponibile per gli utenti di Adobe Analytics per aiutarti a visualizzare i dati delle attività degli utenti direttamente sul tuo sito web. Puoi vedere il numero di clic (o altro [!UICONTROL metrics]) direttamente sui collegamenti della tua pagina.

>[!VIDEO](https://video.tv.adobe.com/v/25451/?quality=12)

Per ulteriori informazioni su questa funzione, consulta la [documentazione](https://marketing.adobe.com/resources/help/en_US/analytics/activitymap/).
