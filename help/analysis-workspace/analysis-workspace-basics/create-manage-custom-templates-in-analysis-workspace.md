# Creare e gestire modelli personalizzati in Analysis Workspace {#create-and-manage-custom-templates-in-analysis-workspace}

Scopri come utilizzare i modelli predefiniti e creare modelli personalizzati in Analysis Workspace, in modo che tu e il team possiate iniziare da un set di rapporti predefinito.

>[!VIDEO](https://video.tv.adobe.com/v/23231/?quality=12)

Analysis Workspace viene fornito con diversi modelli predefiniti che consentono di iniziare subito l’analisi. Detto questo, a volte volete anche solo il vostro modello. Scopri come creare qualsiasi progetto nel tuo modello personalizzato. È possibile aiutare i nuovi utenti a imparare a usare le funzioni più veloci, trovare le dashboard o i progetti preferiti o condividere il dashboard tra i team.

Per ulteriori informazioni sui modelli standard e personalizzati, visita il [documentazione](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/build-workspace-project/starter-projects.html?lang=it).
